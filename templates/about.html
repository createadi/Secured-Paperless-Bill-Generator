“Paperless payment generator with data security” is fully interactive user interface web application for bill generation with great functionalities. This project is developed in order to minimize the use of papers for the simple receipt of purchased items by deploying sms-based payment receipts. It provides functionality to generate bills of products which are listed or even not listed on the web app. It generates the bill and sends a text message to respective customers. This web application also incorporates the data security aspects by involving cryptographic algorithms so that it enables a security layer on the customer’s data.
